## 技术栈
### 前端
  * **Sass** + **Es6**
  * **Vue** + **Vue-cli** + **Vue-router** + **Vuex** + **Mint UI**
### 后端
* **PHP**
## 功能实现
* **首页:** 首页 购物车 分类 我的
* **详情:** 商品详情 评价 讨论 购物车 购买
* **分类**
* **购物车:** 结算
* **我的**
## 使用步骤


``` bash
# 安装依赖
npm install

# 运行打开 localhost:8080
npm run dev

# 编译打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
